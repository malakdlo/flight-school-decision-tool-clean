# Prompt for Gemini: Visual Prototype Request

Create a single-file HTML visual prototype for the Flight School Decision Tool.

Goal:
Build a modern, beginner-friendly aviation assessment funnel that helps users determine whether flight school is realistic for their budget, lifestyle, goals, and eligibility concerns.

Required views:

1. Home page
2. Quiz intro
3. Quiz question flow
4. Loading / analysis state
5. Results dashboard
6. Beginner guide article template

Design direction:

- Modern aviation guidance
- Navy, sky blue, amber palette
- Subtle aviation grid background
- Friendly but realistic tone
- High-conversion landing page structure
- Beginner-friendly, not too technical

Important:
This is a visual/product prototype. Optimize for interaction, layout, and feel. Production code will be refactored later in Astro.
