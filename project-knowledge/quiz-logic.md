# Quiz Logic

## Goal

Match users to one of four aviation paths:

1. Weekend Hobbyist
2. Career Pilot
3. Personal Aircraft / Business Travel
4. Explorer / Not Sure Yet

## Current V1 scoring

Each answer contributes points to one or more result types. The highest score wins. Ties fall back to `explorer` unless there is a clear intent signal from the first question.

## Current question categories

1. Primary aviation goal
2. Biggest concern
3. Weekly time commitment
4. Investment comfort level
5. Desired timeline
6. Medical concerns

## Design principles

- Keep scoring transparent and editable
- Avoid pretending the quiz is a medical or financial authority
- Use quiz answers to guide, not diagnose
- Make all recommendations educational and beginner-friendly

## Future improvements

- Ask country/region
- Ask nearest airport or ZIP/postcode
- Ask age range for career timeline realism
- Ask training frequency
- Add separate parent/teen path
- Add downloadable personalized PDF
