# Product Strategy

## Positioning

A decision-support tool for people considering flight school, built to replace vague internet research with a simple, personalized first-step roadmap.

## Core problem

Aspiring pilots often do not know:

- Which license path fits their goal
- How much training will really cost
- Whether they need a medical certificate
- Whether they should choose Part 61 or Part 141 training
- How often they need to fly
- Whether their age, math skills, or schedule make training unrealistic

## Core product loop

1. User lands on beginner-friendly page
2. User takes realistic assessment
3. User receives path recommendation
4. User enters email for PDF starter kit
5. User receives follow-up education and tools
6. User returns for guides, calculators, and flight school directory

## V1 conversion goal

Primary conversion: quiz completion
Secondary conversion: email capture for Pilot Starter Kit
Tertiary conversion: guide page engagement

## Future monetization options

- Flight school directory leads
- Sponsored school listings
- Paid starter kits or decision tools
- Affiliate aviation gear recommendations
- Parent/teen aviation planning resources
- Localized school comparison pages

## Product principles

- Start with clarity, not complexity
- Give realistic numbers, not hype
- Help the user avoid costly mistakes
- Make next steps concrete
- Build trust before monetization
